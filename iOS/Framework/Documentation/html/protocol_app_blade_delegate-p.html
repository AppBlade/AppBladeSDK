- (void) appBlade: (AppBlade *)  appBlade
applicationApproved: (BOOL)  approved
error: (NSError *)  error 

This method is called when the delegate is notified of whether the Application was approved to run.

Parameters
appBladeThe specific appblade reference the delegate is observing.
approvedThe boolean of whether the application is approved or not.
errorAn optional error parameter.
Returns
This method returns nothing. If false is passed to the approved parameter, A kill() will be sent to the main thread and the app will terminate.

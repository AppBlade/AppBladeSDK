Detailed Description

Utilities for encoding and decoding URL arguments.

Method Documentation

- (NSString *) gtm_stringByEscapingForURLArgument

Returns a string that is escaped properly to be a URL argument.

This differs from stringByAddingPercentEscapesUsingEncoding: in that it will escape all the reserved characters (per RFC 3986 http://www.ietf.org/rfc/rfc3986.txt) which stringByAddingPercentEscapesUsingEncoding would leave.

This will also escape '', so this should not be used on a string that has already been escaped unless double-escaping is the desired result.

- (NSString *) gtm_stringByUnescapingFromURLArgument

Returns the unescaped version of a URL argument.

This has the same behavior as stringByReplacingPercentEscapesUsingEncoding:, except that it will also convert '+' to space.

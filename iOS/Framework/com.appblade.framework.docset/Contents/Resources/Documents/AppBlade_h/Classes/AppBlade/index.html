Introduction

Our main class. AppBlade contains our singleton and all public methods, which are used as entrypoints for the lower level managers.


Groups

Auto Updating

Group members:

-checkForUpdates

Checks with AppBlade anonymously to see if the app can be updated with a new build.

registerWithAppBladePlist

Initial registration method, use before enything else. 

- (void)registerWithAppBladePlist;
Discussion

AppBlade registration that uses the AppBlade.plist that you embedded on setup. It's required that you register before anything else in AppBlade can be used.

registerWithAppBladePlistNamed:

Initial registration method, use before enything else. 

- (void)registerWithAppBladePlistNamed:(NSString*)plistName;
Discussion

AppBlade registration that takes a custom parameter for the plist name that you embedded on setup. This special plist must exist in your main bundle. Note that AppBlade will not find the plist and inject it with tokens if you do this, so this call is not advised.

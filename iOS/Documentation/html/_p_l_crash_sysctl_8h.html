char* plcrash_sysctl_string ( const char *  name)

Wrap sysctl() and fetch a C string, automatically allocating a sufficiently large buffer for the returned data.

Parameters
nameThe sysctl MIB name.
lengthOn success, will be populated with the length of the result. If NULL, length will not be supplied.
Returns
Returns a malloc-allocated NULL-terminated C string containing the sysctl result on success. On failure, NULL is returned and the global variable errno is set to indicate the error. The caller is responsible for free()'ing the returned buffer.
